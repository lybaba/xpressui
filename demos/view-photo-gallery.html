View Photo Gallery

Démonstration de mode="view" avec policy média gallery et données injectées via API runtime.

Live Demo

Points clés

  • Le formulaire est monté normalement puis basculé en view.
  • Les valeurs sont injectées avec setViewValues(...).
  • setFieldMediaPolicy("photos", "gallery") force une grille d’images.
  • Le submit est masqué automatiquement en mode vue.
